## 실시간 PVP대전과 다양한 모드를 지원하는 2048-Master 개발

* **지원하는 OS**
  * Android Mobile
  * Windows
<br>
<br>

* **개발환경**
  * Game Client: Unity (2021.3.1f1 LTS)
  * Game Server: C# (.NET Framework 3.5)
  * Database: Amazon RDS MySQL
